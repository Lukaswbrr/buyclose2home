JSON Maker for BuyClose2Home

This page is for making JSONs for usage of the BuyClose2Home program.

First, here's some explanation:

  • Add Product Default: Adds a product with a main primary key. (Example: computer)
  • Add Product Variation: Adds a product within a main primary key. Example: The computer main primary key has tons of variations. Like a gaming computer, a dell computer, etc

Now, how to use it:

For images, it's recommended that you upload the images to a file management website. I recommend File Garden or Google Drive for uploading images. Remember to mark the file as Unlisted or a way that it's possible to access the image via URL.

Reminder that this is a barebones JSON Maker. Once you create a JSON through this, i recommend editing it with a proper JSON editor. This is mainly for getting templates ready.
